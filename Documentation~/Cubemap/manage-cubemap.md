# Manage generated cubemaps

Organize and manage generated cubemaps using the **Generations** panel.

The **Generations** panel stores all the generated cubemaps and provides tools to preview, promote, and reuse them.

To manage cubemaps, follow these steps:

1. Right-click a generated cubemap to open the context menu.
2. Choose from the following options:

   | Option | Description |
   | ------ | ----------- |
   | **Select** | Highlights the cubemap in the **Project** window. |
   | **Promote to current asset** | Replaces the current cubemap with the selected one. |
   | **Promote to new asset** | Creates a new asset from the selected cubemap. |
   | **Reveal in Finder** | Opens the file location on your computer. |
   | **Show Generation Data** | Displays the model and prompt details used to create the cubemap. |

      The **Generation Data** window lets you view and reuse settings used to create a generated cubemap.

      | Option | Description |
      | ------ | ----------- |
      | **Use All** | Copies all settings (model, prompt, and reference images) to the **Generation** area. |
      | **Use** | Copies only the selected settings to the **Generation** area. |
      | **Copy** | Copies the settings and applies them to generate a new cubemap. |
